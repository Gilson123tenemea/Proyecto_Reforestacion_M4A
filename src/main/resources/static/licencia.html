Términos y Condiciones de Uso

Esta licencia establece las condiciones bajo las cuales se puede utilizar el software desarrollado para el proyecto de reforestación del Parque Nacional El Cajas. El uso del software implica la aceptación de estos términos.

1. Uso permitido

El software solo podrá ser utilizado por los usuarios debidamente registrados y autorizados, dentro del marco del proyecto de reforestación. El acceso y uso del software están restringidos a las actividades relacionadas con la reforestación y el seguimiento de intervenciones, tal como se define en el alcance del proyecto.

2. Restricciones

No se permite la modificación, distribución, reproducción, ni el uso comercial del software sin la autorización expresa de los administradores del proyecto. Cualquier intento de manipular o utilizar el software fuera del contexto del proyecto de reforestación será considerado un incumplimiento de los términos de uso.

3. Seguridad

El usuario se compromete a mantener la seguridad del sistema, proteger la integridad de la información almacenada y a no divulgar ni compartir información confidencial relacionada con el proyecto, los participantes o las actividades realizadas, excepto en los casos permitidos por la normativa del proyecto.
