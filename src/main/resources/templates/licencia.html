Licencia del Software

Este sistema se publica bajo una licencia de software propietario, lo que garantiza un control completo sobre su distribución y modificación. La decisión de optar por esta licencia se ha tomado con base en las necesidades específicas del proyecto Llakta Forest, el cual tiene como objetivo principal la reforestación colaborativa en la región del Cajas.

Características de la Licencia Propietaria

  • Control de uso: El uso del software está restringido y no podrá ser distribuido ni modificado sin autorización previa.
  • Acceso restringido al código fuente: El código fuente no estará disponible públicamente.
  • Soporte y mantenimiento: El software se mantendrá y soportará de forma controlada, asegurando que cumple con los requerimientos del proyecto Llakta Forest.

Justificación de la Elección de la Licencia

  • Seguridad y control: Mantener el control sobre el uso y distribución es crucial para la gestión de actividades de reforestación.
  • Protección de datos: La licencia propietaria minimiza los riesgos de exposición de datos sensibles relacionados con el proyecto.
  • Adaptación a necesidades específicas: Permite realizar adaptaciones o mejoras sin la obligación de compartir las modificaciones con terceros.

Consideraciones Legales en Ecuador

En Ecuador, la Ley de Propiedad Intelectual protege los derechos de autor, lo que garantiza que la propiedad intelectual del software sea protegida. Es fundamental que se registre el software en la Dirección Nacional de Derechos de Autor y que se establezcan acuerdos claros sobre el uso del software.

Conclusión

La adopción de una licencia de software propietario para el sistema del proyecto Llakta Forest asegura un control adecuado sobre el uso y distribución del software. Esta decisión se alinea con las necesidades específicas del proyecto y con las leyes ecuatorianas de propiedad intelectual.
